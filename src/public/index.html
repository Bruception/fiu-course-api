Combined Queries

                

                    Let's say we are trying to find all courses whose subject starts with J, code starts with 1, and is 5 units.
                    Here is an example of how you might query for this.    
                

                
GET /api?subject=J&code=1&units=5

                

            

            

                
Multi-value Queries

                

                    Let's say we are trying to find all labs whose subject starts with AST or COP.
                    Here is an example of how you might query for this.    
                

                
GET /api?subject=AST&subject=COP&isLab

                

            

            

                
Data Format

                

                    You can specify how the returned data is serialized with the format parameter.
                    Currently this API only supports the following formats:
                    
    
                        
  • JSON
    • 
                        
  • Text
    • 
                        
  • XML
    • 
                        
  • YAML
    • 
                    

                    Here is an example of how you might query your data in YAML.
                

                
GET /api?format=yaml

            

            

                
Keyword Search

                

                    You can fuzzy search for courses by their name and description with the keywords parameter.
                    The keywords parameter is a string where the words are space-delimited.
                    This API will reject any string with more than 5 keywords.
                    Here is an example of a query using the keywords parameter:
                

                
GET /api?keywords=electric theories magnets
